2. 문제의 난이도는 다음과 같이 제한한다
    1. 백준  :  Silver5 - Gold1
    2. 프로그래머스  :  Lv1 - Lv3
    3. SWEA  :  D3 - D5
    4. 코드트리  :  기출문제 / 삼성 SW 역량 테스트
3. 선정한 문제는 일요일 자정까지 Notion 및 Github 위키에 작성한다
